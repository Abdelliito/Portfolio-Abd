This is my personal portfolio website built using React, Vite, and Tailwind CSS.
It showcases my skills, projects, and contact details in a modern, responsive UI.

Features
- Fully responsive design
- Smooth navigation to Home, About, Projects, and Contact sections
- Modern UI with Tailwind CSS
- Mobile-friendly hamburger menu
- Loading animation screen
- Clean and lightweight code

Technologies Used
- React: Frontend framework
- Vite: Development & build tool
- Tailwind CSS: Styling and layout
- JavaScript (ES6): Logic & interactivity

Installation & Setup
1. Clone the repository
   git clone <your-repository-link>

2. Go to project folder
   cd portfolio

3. Install dependencies
   npm install

4. Start development server
   npm run dev


Build for Production
npm run build
